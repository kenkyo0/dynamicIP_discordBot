# DynamicIP_DiscordBot

This is a light-weight Discord Bot for Servers, which updates you about the current **IPv4/IPv6** address of your Server.
Perfect for a Minecraft-Server, if you don't want to buy a Domain-Name.

## Requirements
- dpp libary
 - 
- cmake
 - 
- build-essential

## Installation

```bash
git clone ...
cd dynamicIP_ServerBot/cmake
cmake ..
```
